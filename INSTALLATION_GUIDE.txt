STEP 1: SETUP WEB SERVER
================================================================================

1. Install XAMPP/WAMP/LAMP on your computer
2. Start Apache and MySQL services

================================================================================
STEP 2: CREATE DATABASE
================================================================================

Method 1: Using phpMyAdmin
--------------------------
1. Open browser and go to: http://localhost/phpmyadmin
2. Click on "Import" tab
3. Click "Choose File" and select:
   /path/to/webapp/database/classroom_booking_system.sql
4. Click "Go" button at the bottom
5. Wait for success message

Method 2: Using MySQL Command Line
----------------------------------
1. Open command prompt/terminal
2. Navigate to MySQL bin folder
3. Run: mysql -u root -p
4. Enter MySQL password (usually empty for XAMPP)
5. Run: source C:/xampp/htdocs/webapp/database/classroom_booking_system.sql
6. Exit MySQL: exit;

================================================================================
STEP 3: CONFIGURE DATABASE CONNECTION
================================================================================

1. Open: config/database.php
2. Update these values if needed:

   define('DB_HOST', 'localhost');     // Usually 'localhost'
   define('DB_USER', 'root');          // MySQL username
   define('DB_PASS', '');              // MySQL password (empty for XAMPP)
   define('DB_NAME', 'classroom_booking_system');

================================================================================
STEP 4: DEPLOY PROJECT FILES
================================================================================

1. Copy the 'webapp' folder to your web server directory:

   XAMPP Windows: C:\xampp\htdocs\webapp
   XAMPP Linux: /opt/lampp/htdocs/webapp
   WAMP: C:\wamp\www\webapp
   LAMP: /var/www/html/webapp

2. Ensure all files are readable by the web server

================================================================================
STEP 5: CONFIGURE APPLICATION URL
================================================================================

1. Open: config/config.php
2. Update SITE_URL constant:

   define('SITE_URL', 'http://localhost/webapp');

   If your project is in a different folder:
   define('SITE_URL', 'http://localhost/your-folder-name');

================================================================================
STEP 6: ACCESS THE APPLICATION
================================================================================

1. Open your web browser
2. Navigate to: http://localhost/webapp
3. You should see the home page

TROUBLESHOOTING
================================================================================

Problem: "Connection failed" error
Solution: 
- Check if MySQL service is running
- Verify database credentials in config/database.php
- Ensure database is created successfully

Problem: Blank page or errors
Solution:
- Check PHP error logs
- Ensure PHP version is 7.4 or higher
- Check file permissions (should be readable)

Problem: CSS not loading
Solution:
- Check SITE_URL in config/config.php
- Clear browser cache
- Check assets folder is accessible

Problem: Session errors
Solution:
- Check if session_start() is working
- Ensure tmp directory has write permissions
- Clear browser cookies
